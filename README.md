# Pavan Kumar - Personal Portfolio Web Application

Welcome to the official personal portfolio repository of **Pavan Kumar**, Senior Data Engineer & BI Specialist with 6+ years of experience building robust ETL/ELT pipelines, designing high-performance data models, and delivering executive-level business intelligence.

This web application features a modern, responsive user interface designed with pure HTML, Vanilla CSS, and modern interactive JavaScript.

---

## 🌟 Key Sections

- **Hero Section**: Quick overview, interactive roles typing animation, resume download link, and direct social contacts.
- **About Me**: Professional background spanning 6+ years in healthcare and banking data analytics, highlighting core competencies and quantifiable statistics.
- **Skills & Tools**: Grid display of capabilities across Databases & SQL (MS SQL Server, Azure SQL), ETL Pipelines (ADF, SSIS, Databricks), BI Tools (Power BI, DAX), and Languages (Python, Pandas, NumPy).
- **Professional Experience**: Interactive vertical timeline detailing achievements and roles at **Cognizant** and **Happay**.
- **Projects Showcase**: Interactive portfolio cards with detailed modal views for enterprise data pipelines, BI dashboards, and analytics models.
- **Contact**: Interactive contact form and direct email/phone details.

---

## 🛠️ Technology Stack

- **Frontend Core**: Semantic HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Styling & Icons**: Custom design system with glassmorphism, glowing CSS effects, Google Fonts (*Inter*, *Manrope*, *Fira Code*, *Caveat*), and Bootstrap Icons.
- **Animations**: CSS keyframes, interactive modals, and custom JavaScript scroll-reveal observers.

## 🚀 Getting Started

No external server setup or build tools are required to run this application.

1. Clone the repository:
   ```bash
   git clone https://github.com/pavankumarM1998/Personal-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Personal-Portfolio
   ```
3. Open `index.html` directly in any web browser.
